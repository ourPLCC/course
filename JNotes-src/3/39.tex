IfExp
%%%
    public Val eval(Env env) {
        Val v = testExp.eval(env);
        if (v.isTrue())
            return trueExp.eval(env);
        else
            return falseExp.eval(env);
    }
%%%
\end{qv}
Notice how we use the \verb'isTrue()' boolean method applied
to an instance of \verb'Val'.\exx
